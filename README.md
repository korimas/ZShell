# ZShell
A simple session manager for putty.

download: https://github.com/zpdev/ZShell/releases

## 颜色字体配置
请进入软件目录的resources目录，打开putty.exe，执行以下操作：
1. Load DefaultSession
2. 编辑颜色字体等配置
3. Save到Default Session。
新打开的tab页面就会使用新的设置。
